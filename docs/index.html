CLAUDE MEMORY

Persistent Memory for AI Agents

Claude Memory gives your AI agent the gift of persistent memory. No longer will your AI forget the lessons of yesterday. Every decision recorded, every failure remembered, every pattern preserved across sessions.

Get Started

Place the Setup Guide in your project root and tell Claude: "Read Setup.md and install Claude Memory." Claude will handle the installation automatically.

The Uninstall Guide provides instructions for removing Claude Memory if needed.

Key Features

Inscribed Memories

Decisions, patterns, warnings, and learnings persist in SQLite with Qdrant vectors, surviving beyond session death.

Semantic Sight

Vector embeddings grant deep understanding. Claude Memory finds connections invisible to mortal keyword search.

Causal Chains

Graph memory traces the WHY. Follow the chain of decisions that led to your current code state.

Sacred Laws

Rules engine enforces project constraints. Must-do, must-not, and warnings guard against forbidden paths.

Failure Amplification

Failed approaches receive 1.5x relevance. Claude Memory ensures past mistakes haunt future attempts.

Temporal Decay

Episodic memories fade over 30 days. Semantic truths remain eternal. Claude Memory knows what matters.

Enforcement Wards

Git pre-commit hooks block commits when discipline falters. No escape for the undisciplined. Claude Memory enforces.

The Inner Eye

Through arcane parsing, Claude Memory perceives the bones of your code. Classes, functions, methods—all laid bare to its gaze.

Eternal Vigil

Claude Memory never sleeps. When files bearing memories are touched, whispers echo from the abyss. No change escapes notice.

The Deep Archive

In the Qdrant depths, meaning dwells beyond mere words. Hybrid search pierces the veil between keyword and intent.

The Unconscious Scribe

Passive capture inscribes memories without command. Hooks whisper to Claude Memory as you work, recording decisions unbidden.

The Compressed Grimoire

Endless Mode condenses wisdom to its essence. When context grows thin, memories compress—75% smaller, meaning preserved.

The Coven Bond

Linked projects share knowledge across repositories. Frontend knows backend's warnings. The coven remembers as one.

The Contradiction Ward

New decisions are tested against past failures. Claude Memory warns when you walk a path already proven ruinous.

The Keen Eye

Tags infer themselves from memory content. "fix" begets bugfix, "cache" summons perf. Claude Memory reads between lines.

The Swift Watcher

Only changed files are reindexed. SHA256 hashes mark what Claude Memory has seen. Time saved, nothing missed.

True Names

Entities bear their full lineage: module.Class.method. Move code freely—stable IDs survive the journey.

The Balanced Scale

Tune the search between keyword and meaning. Adjust hybrid weight to match your domain's tongue.

Quick Start

For those who prefer the command line: 

git clone https://github.com/hughryan/claude-memory.git ~/claude-memory
pip install -e ~/claude-memory
claude mcp add claude_memory --scope user -- python -m claude_memory.server
python -m claude_memory.cli install-hooks  # Install git hooks
python -m claude_memory.cli index          # Index code
claude  # Restart to activate

Windows requires additional steps. See the full Setup Guide for details.

The Protocol

For best results, follow this recommended workflow: 

SESSION DAWN       get_briefing()         # Commune first
BEFORE CHANGES     context_check()        # Seek counsel
AFTER DECISIONS    remember()             # Inscribe the choice
AFTER COMPLETION   record_outcome()       # Seal with fate
CAUSAL INQUIRY     trace_chain()          # Understand WHY
CODE SIGHT         find_code()            # Search by meaning
IMPACT VISION      analyze_impact()       # See consequences
AT COMMIT TIME     pre-commit hooks       # Enforcement wards
